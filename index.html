Login

This login is a privacy gate for your public GitHub Pages site. Real security comes from protecting your data API (we do that with an API key in Settings).

How “secure” is this?

GitHub Pages is public. Any “client-side login” can be bypassed by someone technical. The important part is that your Google Sheet API endpoint requires an API key that is stored only on your devices (browser localStorage), not in this repo.
